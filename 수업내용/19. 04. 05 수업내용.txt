재귀메소드
메소드 오버로딩
 - 한 클래스에서 동일한 메소드명을 가진 메소드가 여러개인 경우
 - 조건
 1. 매개변수의 갯수가 다른 경우
 2. 매개변수의 갯수가 같지만 자료형이 다른 경우

클래스 = 멤버변수, 멤버메소드
멤버변수 : 일반적으로 접근제한자를 private으로
멤버메소드 : 일반적으로 접근제한자를 public으로
멤버변수의 값을 변경하려면 멤버 메소드를 이용하여 변경한다.

접근제한자
public : 전부(아무나)
private : 본인
default : 같은 패키지
protected : 본인 + 자식(파생)클래스

public > default > protected > private 
------------------------------------------------------------------------


클래스 멤버 vs 객체 멤버
-객체 멤버
객체 멤버는 객체가 생성된 후에 사용할 수 있다.
객체.멤버변수, 객체.멤버메소드로 호출한다.
객체 멤버는 만들어진 객체마다 가지고 있다.

-클래스 멤버
객체의 생성 없이 사용 할 수 있다.
클래스.멤버변수, 클래스.멤버메소드로 호출한다.
클래스 멤버는 하나씩만 만들어진다.

클래스 멤버 메소드에 클래스 멤버 변수는 사용 가능하지만
객체 멤버 변수는 사용 할 수 없다.

getter 와 setter
멤버변수가 private일 경우 해당 멤버 변수 값을 확인 할 수 있는
메소드가 필요하고 그 메소드를 getter, setter라 한다.
setter는 해당 멤버변수의 값을 변경하는 기능을 하고
getter는 해당 멤버변수의 값을 가져오는 기능을 한다.
